# react-native-native-qr-display

Native JSI QR Code Display for React Native

## Installation

```sh
npm install react-native-native-qr-display
```

## Usage

```js
import { multiply } from "react-native-native-qr-display";

// ...

const result = await multiply(3, 7);
```
